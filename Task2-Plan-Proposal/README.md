# Task 2: Software Development Proposal

## 1. Overview
This proposal outlines the development of a secure web-based dashboard for Daikibo Industrials to monitor the operational health of manufacturing equipment across four factories. The dashboard will provide real-time visibility into the status of 36 machines (9 machines per factory) using telemetry data collected from the company's IIoT infrastructure.

The solution will be accessible exclusively through Daikibo's internal intranet and will integrate with the company's existing authentication system, allowing employees to securely access the platform using their organization-wide credentials. The dashboard will provide an intuitive interface for viewing current machine status and historical status information.

## 2. Scope
The proposed solution consists of a single-page dashboard application designed for internal use.

### Features
* Secure access through Daikibo's intranet.
* Integration with the internal authentication server for user login.
* Display of all four factories and their monitored devices.
* Real-time visualization of the current health status of each machine.
* Collapsible and expandable factory-level views.
* Collapsible and expandable device-level views.
* Historical status display for individual machines.
* Clear and user-friendly interface based on the provided wireframe.
* Responsive design for use on standard desktop devices within the organization.

### Dashboard Structure
**Factory Level**
* Factory Name
* Overall Factory Status
* Expand/Collapse Control

**Device Level**
* Device Name/Identifier
* Current Health Status
* Expand/Collapse Control

**Historical Status View**
* Previous Status Records
* Timestamp Information
* Status Trend Visibility
