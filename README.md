## 🛠️ Tasks Overview

### [Task 1: Telemetry Data Conversion](./Task1-Telemetry-Conversion/)
**Goal:** Standardize telemetry data from various factory devices into a unified JSON schema.
- **What we did:** Developed a Python script to parse two distinct incoming JSON formats, extracted necessary fields like `deviceID`, `timestamp`, and `location` parameters, and transformed them into a consistent structure. The solution includes `unittest` cases to ensure format conversions are accurate and robust against missing fields.

### [Task 2: Software Development Proposal](./Task2-Plan-Proposal/)
**Goal:** Draft a formal development proposal for a private intranet dashboard to monitor machine health across factories.
- **What we did:** Created a structured proposal detailing the project overview, scope (including secure authentication and real-time visualization features), estimated man-hours, a concrete development timeline, and a continuous support plan. This proposal serves as a blueprint for the client's internal engineering team.
